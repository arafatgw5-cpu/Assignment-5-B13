1 **var, let, const:** `var` is function-scoped and can be redeclared, `let` is block-scoped and can be updated but not redeclared, `const` is block-scoped and cannot be updated or redeclared.
2 **Spread operator (...):** Expands an array or object into individual elements or properties.
3 **map(), filter(), forEach():** `map()` returns a new array, `filter()` returns a filtered array, `forEach()` executes a function for each element without returning a new array.
4 **Arrow function:** A concise syntax for functions using `()=>`, often without its own `this`.
5 **Template literals:** Strings using backticks `` ` `` allowing interpolation `${}` and multi-line strings.
